# Experiment 3: Regression Tree Using Income Dataset

## Objective

To implement a Decision Tree Regression algorithm using the Income Dataset and predict salary based on demographic and educational attributes.

---

## Description

Decision Tree Regression is a supervised machine learning technique used to predict continuous numerical values. The algorithm recursively partitions the dataset into smaller subsets and generates prediction rules in the form of a tree structure.

In this experiment, a regression tree model is developed to estimate salary based on age and educational qualification.

---

## Dataset

The dataset contains information related to individuals, including:

* Age
* Education
* Salary

### Target Variable

* **Salary**

### Input Features

* Age
* Education

---

## Methodology

### Step 1: Data Exploration

* Load the dataset using Pandas.
* Examine dataset structure and records.
* Visualize relationships using boxplots and scatter plots.

### Step 2: Data Preprocessing

* Separate input features and target variable.
* Convert categorical attributes into numerical format using one-hot encoding.

### Step 3: Train-Test Split

* Divide the dataset into training and testing subsets.
* Use 60% training data and 40% testing data.

### Step 4: Model Training

* Train a Decision Tree Regressor using the training dataset.

### Step 5: Prediction and Evaluation

* Predict salary values for test data.
* Evaluate model performance using:

  * Mean Absolute Error (MAE)
  * R² Score

### Step 6: Tree Visualization

* Visualize the regression tree structure.
* Interpret decision rules generated by the model.

### Step 7: Feature Importance Analysis

* Determine the relative importance of each input feature.

### Step 8: Cost Complexity Pruning

* Generate pruning path using Cost Complexity Pruning.
* Evaluate multiple models with different alpha values.
* Select the optimal alpha value based on test performance.

### Step 9: Final Model Development

* Train the pruned regression tree using the best alpha value.
* Compare training and testing performance.
* Visualize the optimized regression tree.

## Performance Metrics

### Mean Absolute Error (MAE)

Measures the average magnitude of prediction errors.

### R² Score

Measures how well the model explains the variation in the target variable.

A higher R² score indicates better predictive performance.
